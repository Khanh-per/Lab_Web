# Shopping Cart Exercise

This exercise implements a dynamic shopping cart system using HTML, CSS, and JavaScript. Users can add items to cart, adjust quantities, and see real-time price updates.

## Features

- Dynamic cart updates
- Real-time price calculations
- Quantity adjustment controls
- Running total display
- Responsive design

## Cart Functionality

1. **Product Display**
   - Product name
   - Individual price
   - Quantity selector
   - Subtotal per item

2. **Cart Operations**
   - Add items to cart
   - Remove items from cart
   - Adjust quantities
   - Real-time total calculation

3. **Price Calculations**
   - Individual item subtotals
   - Cart total
   - Updates automatically with quantity changes

## User Interface Features

- Intuitive quantity controls (+ and - buttons)
- Clear visual feedback for price updates
- Responsive design for various screen sizes
- Easy-to-read layout

## How to Use

1. Open `Shopping-Cart.html` in a web browser
2. Browse the available products
3. Use + and - buttons to adjust quantities
4. Watch subtotals and total update automatically
5. Remove items using the remove button

## Implementation Details

- Pure JavaScript implementation
- Real-time price updates
- No page reload required for updates
- Local cart state management
- Event delegation for dynamic elements

## Cart Logic

- Automatic quantity validation
- Prevents negative quantities
- Real-time price calculations
- Proper number formatting for currency
